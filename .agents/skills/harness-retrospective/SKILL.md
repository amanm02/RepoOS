# Harness Retrospective Skill

## Goal

Convert failure evidence into a concrete harness improvement.

## Process

1. Reconstruct what happened.
2. Identify root cause.
3. Identify the missing guardrail.
4. Choose the smallest fix surface.
5. Add or propose a regression case.
6. Update the improvement backlog.

## Fix surfaces

- Test
- Hook
- Skill
- Doc update
- Schema change
- MCP change
- Subagent update
- Folder restructure

## Output

- Failure summary
- Root cause
- Missing guardrail
- Selected fix
- Regression case
- Follow-up issue
