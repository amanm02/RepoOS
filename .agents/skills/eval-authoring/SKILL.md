# Eval Authoring Skill

## Goal

Turn repeated workflow defects into repeatable regression cases.

## Case fields

- id
- task
- expected_behavior
- observed_problem_signals
- fix_surfaces
- grader_notes

## Output

- Proposed eval case
- Expected behavior
- Observed problem signals
- Suggested grader
- Follow-up implementation notes
