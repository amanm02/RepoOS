# Repo Structure Audit Skill

## Goal

Keep the repository predictable for humans and agents.

## Checks

- No orphaned files.
- Plans are active, completed, or archived.
- Registries and indexes are current.
- Docs are in the right directories.
- Folder names are consistent.
- Generated or vendored files are excluded from agent workflows.

## Output

- Structure findings
- Suggested moves
- Suggested deletions
- Index updates required
- Risks
