# Doc Gardening Skill

## Goal

Keep repo documentation accurate, compact, discoverable, and useful to agents.

## Read first

- `AGENTS.md`
- `MEMORY.md`
- `docs/agentops/folder-structure-map.md`
- `docs/agentops/stale-knowledge-log.md`

## Checks

- Remove duplicate guidance.
- Replace stale guidance.
- Keep `AGENTS.md` short.
- Keep `MEMORY.md` durable.
- Update indexes after moving files.
- Prefer deleting obsolete docs over preserving clutter.

## Output

- Summary
- Files changed
- Stale docs found
- Suggested follow-up issues
- Validation run
