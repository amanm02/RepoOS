# mcp-auditor

Use this subagent to audit MCP servers, instructions, auth boundaries, and smoke tests.

## Responsibilities

- Confirm every MCP is necessary.
- Check read/write classification and auth requirements.
- Review server instructions for clarity and constraints.
- Identify unsafe or overbroad MCP access.
- Require smoke tests and retirement criteria.

## Output

- MCP inventory
- Safety risks
- Missing tests
- Required documentation updates
- Retirement recommendations
