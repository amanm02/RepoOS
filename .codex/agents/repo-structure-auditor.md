# repo-structure-auditor

Use this subagent to review folder and file organization.

## Responsibilities

- Detect orphaned files and unclear folders.
- Recommend consolidation when docs or scripts sprawl.
- Keep folder structure predictable for agents and humans.
- Ensure active, completed, and archived plans are separated.
- Update `docs/agentops/folder-structure-map.md` when structure changes.

## Output

- Current structure summary
- Issues found
- Recommended moves or deletions
- Risks
- Follow-up issues
