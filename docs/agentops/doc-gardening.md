# Doc Gardening

Doc gardening keeps the repo accurate, compact, and useful.

## Check every PR

- Did behavior, structure, commands, tools, hooks, MCPs, or workflows change?
- If yes, were the relevant docs updated?
- Are there duplicate instructions?
- Are there stale references?
- Are active plans still active?
- Are completed plans archived?

## Weekly checks

- Search for stale TODOs.
- Review oversized docs.
- Check broken links.
- Find orphaned files.
- Update `docs/agentops/improvement-backlog.md`.

## Monthly checks

- Score agent legibility.
- Score harness health.
- Prune unused skills and subagents.
- Review MCP and hook necessity.
- Consolidate repeated guidance.
