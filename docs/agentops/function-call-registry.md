Use this registry for structured function-call contracts.

## Requirements

- Inputs are explicit and narrow.
- Outputs are structured and documented.
- Calls have examples.
- Validation covers required fields and common edge cases.
- Programmatic consumers should not depend on free-form prose when a schema is possible.
